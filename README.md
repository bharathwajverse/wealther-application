# atmosphere-weather

A modern weather forecasting web application built with Next.js, React, Tailwind CSS, and the Open-Meteo API.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## GitHub

This repo includes the required GitHub support files:
- `package.json`
- `.gitignore`
- `README.md`

## Project structure

- `app.js`, `index.js`, `script.js` — main app logic
- `style.css` — styling
- `src/` — application source folders
- `weather-service.js` — weather API handling
